# Getting Started with MarkScout

## First Launch

When you first open MarkScout, you'll see an onboarding screen suggesting common directories to watch. Pick one or browse for a custom folder.

**Recommended paths for vibe coders:**
- `~/.claude/` — Claude Code configuration and memory files
- `~/Documents/` — General documents
- Your main project directory

## Navigating

Use the keyboard for fast navigation:

- **j/k** — move up/down through files
- **1/2/3/4** — switch sidebar views (New, Recents, Folders, Favorites)
- **/** — jump to search
- **s** — star the current file
- **?** — show all shortcuts

## Typography

Open Preferences (`Cmd + ,`) and choose a typography preset:

- **Classic** — serif headings and body, best for general reading
- **Modern** — clean system fonts, minimal
- **Literary** — elegant serif throughout
- **Developer** — monospace everything, for code docs
- **Accessible** — larger text, wider spacing

## Color Palettes

Click the palette dots at the top of the reader to switch between 12 color themes. There are 9 dark themes and 3 light themes.

## Smart Collections

At the bottom of the sidebar, expand "Collections" to see your files auto-grouped by type: Plans, Architecture, Requirements, Research, and more.
